Important FAQ

What is the Document Object Model(DOM) ?

The Document Object Model (DOM) is a platform and language-neutral interface that allows programs and scripts to dynamically access and update the content, structure, and style of a document

How do you select an element from DOM ?

There are many way to select an element from DOM . among them two are popular one is Using "getElementById" it allows us to select an element by its ID. Another "getElementsByClassName" select an element by its ClassName.

What is event Delegation in the context of DOM , and why is it useful ?

It is an event-handling pattern that allows you to handle events at a higher level in the DOM tree other than the level where the event was first received.

How do you manipulate an element's attributes and styles using the DOM ?

You can manipulate an element's attributes using methods like getAttribute(), setAttribute(), createElement(), and modify styles via the style property or by adding/removing CSS classes. This allows dynamic changes to an element's behavior and appearance
